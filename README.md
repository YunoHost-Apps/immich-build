# immich-build

Scripts for building and binary releases of [immich](https://github.com/immich-app/immich) for [YunoHost package](https://github.com/YunoHost-Apps/immich_ynh).

## Building

Prebuilt are currently produced and published with the workflow [build_immich.yml](https://github.com/YunoHost-Apps/immich-build/blob/main/.github/workflows/build_immich.yml)
